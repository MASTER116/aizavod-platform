# KOMPAS-QC: Product Requirements Document
## AI-конвертер китайских чертежей (GB) в российский ГОСТ для КОМПАС-3D

**Версия:** 1.0
**Дата:** 2026-03-24
**Автор:** AI Zavod / Khalyapov

---

## 1. Проблема

Каждая единица промышленного оборудования из Китая ($5.8B импорт в 2024, +35% YoY) **обязана** иметь документацию по ГОСТ/ЕСКД для:
- Таможенного оформления
- Сертификации (ТР ТС 010/2011)
- Установки и эксплуатации
- Проверок Ростехнадзора

**Текущее решение:** ручная конвертация инженерами + переводчиками.
- Стоимость: 30 000 — 100 000 руб. за комплект документации
- Сроки: 2-4 недели
- Ошибки: частые (особенно в маппинге марок стали)

**Автоматизированных решений: 0.** Ни один инструмент на рынке не конвертирует GB→ГОСТ автоматически.

---

## 2. Решение

**KOMPAS-QC** — AI-инструмент, который:
1. Принимает чертёж в формате DXF/DWG/STEP (экспорт из китайского CAD)
2. Распознаёт элементы GB-стандарта (основная надпись, материалы, допуски, шероховатость, сварка)
3. Автоматически конвертирует в ГОСТ/ЕСКД
4. Генерирует готовый чертёж для КОМПАС-3D (.cdw) или DXF с ГОСТ-оформлением

**Ценностное предложение:** 2-4 недели ручной работы → 15 минут автоматической конвертации.

## 5. Функциональные требования

### MVP (v0.1) — 4-6 недель

#### F1: Парсер DXF-чертежей
- Чтение DXF файлов (экспорт из китайского CAD)
- Извлечение текстовых блоков, размеров, таблиц
- Распознавание китайского текста (OCR если растр)
- **Библиотека:** `ezdxf` (Python)

#### F2: Маппинг марок стали GB→ГОСТ
- База соответствий 50+ основных марок:
  - Q235B → Ст3сп
  - Q355B → 09Г2С
  - 45 → Сталь 45
  - 40Cr → 40Х
  - 06Cr19Ni10 → 08Х18Н10
  - HT200 → СЧ20
  - QT400-18 → ВЧ40
- AI-валидация: проверка что маппинг корректен по хим. составу
- Обработка нестандартных/неизвестных марок (флаг для ручной проверки)

#### F3: Конвертация основной надписи
- GB/T 10609.1 → ГОСТ 2.104-2006
- Перевод полей: 零件名称 → Наименование, 材料 → Материал, 比例 → Масштаб
- Генерация рамки и штампа по ГОСТ

#### F4: Конвертация обозначений шероховатости
- GB/T 131-2006 (ISO 1302) → ГОСТ 2.309-73
- Замена символов на ГОСТ-вариант

#### F5: Конвертация обозначений сварки
- GB/T 324-2008 → ГОСТ 2.312-72
- Замена ссылок на стандарты: GB/T 985.x → ГОСТ 5264/14771/8713

#### F6: Перелитеровка баз GD&T
- Latin (A, B, C) → Cyrillic (А, Б, В)

#### F7: Генерация результата
- Выходной DXF с ГОСТ-оформлением
- PDF-отчёт о конвертации (что изменено, что требует ручной проверки)
- JSON-лог для аудита

### Web UI (v0.2) — +2 недели
- Загрузка файла через браузер
- Предпросмотр до/после
- Редактирование маппингов вручную
- Скачивание результата

### КОМПАС-3D плагин (v0.3) — +4 недели
- COM API интеграция
- Кнопка "Конвертировать GB→ГОСТ" в интерфейсе КОМПАС
- Прямая работа с .cdw файлами
- Автозаполнение штампа

### Telegram бот (v0.2) — +1 неделя
- Отправить DXF → получить сконвертированный DXF + PDF-отчёт
- Интеграция с CONDUCTOR

## 7. Техническая архитектура

```
[Telegram Bot / Web UI / КОМПАС Plugin]
              |
      [FastAPI Backend]
              |
    [KOMPAS-QC Engine]
   /      |       |      \
[DXF Parser] [Material Mapper] [Symbol Converter] [Title Block Generator]
   |           |                    |                      |
[ezdxf]  [GB-GOST DB]     [GOST Symbol Lib]    [GOST 2.104 Template]
              |
      [AI Validator (Ollama/Claude)]
              |
      [CONDUCTOR Integration]
```

### Стек
- **Backend:** Python 3.11, FastAPI
- **DXF:** ezdxf (чтение/запись DXF)
- **DWG:** ODA File Converter (конвертация DWG→DXF)
- **AI:** Claude Haiku (валидация) / Ollama zavod-opus (локально)
- **DB:** PostgreSQL (история конвертаций, маппинги)
- **КОМПАС:** win32com (для плагина, только Windows)

## 10. Грантовая стратегия

### Цепочка грантов
1. **УМНИК** (сейчас) — 500K руб., proof of concept
2. **Старт-ИИ-1 очередь 3** (Q3-Q4 2026) — до 5M руб., полная разработка
3. **РФРИТ** (сентябрь 2026) — крупный грант, нужно ООО + 50% софинансирование
4. **Старт-2** (2027) — до 8M руб., масштабирование

### Соответствие госприоритетам
- Импортозамещение (адаптация китайской документации к РФ стандартам)
- Технологический суверенитет (снижение зависимости от ручного перевода)
- Программа "Новое индустриальное ПО" Минцифры
- Дорожная карта развития САПР (АСКОН — ключевой участник)

### Для заявки акцентировать
- **Нулевая конкуренция** — нет аналогов на рынке
- **Обязательный спрос** — регуляторное требование (ТР ТС)
- **Растущий рынок** — импорт из Китая +35% YoY
- **Интеграция с КОМПАС-3D** — российский CAD, АСКОН как потенциальный партнёр
