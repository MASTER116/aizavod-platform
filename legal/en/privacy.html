1. General Provisions

1.1. This Privacy Policy ("Policy") governs the processing and protection of personal data of AI ZAVOD platform users ("Platform").

1.2. The data controller is the person administering the Platform ("Operator").

1.3. This Policy is developed in accordance with Federal Law No. 152-FZ "On Personal Data" (Russian Federation) and other applicable legislation.

4. AI Data Processing

4.1. The Platform uses machine learning models running on the Operator's local infrastructure (including open-source models via Ollama).

4.2. User data submitted to AI agents is not sent to external third-party servers for language model processing. All computations are performed on the Operator's servers.

4.3. The self-learning module (DARWIN) uses anonymized and aggregated data only. Personal data is not used in model training.

4.4. AI agent interaction logs are retained for up to 90 days for quality assurance and dispute resolution.

5. Data Security

5.1. The Operator implements the FORTRESS multi-layer security system:

  • Layer 1: Network security -- GeoIP filtering, SSH hardening, firewalling
  • Layer 2: Application security -- injection protection, input validation, access control
  • Layer 3: Deterministic control -- critical data operations use verified code without AI models

5.2. Security design follows OWASP Agentic Top 10 recommendations.

5.3. Additional measures include TLS/SSL encryption, encryption at rest, role-based access control, regular backups, and security event monitoring.

6. Cross-Border Data Transfer

6.1. User data is stored and processed on servers located in the Russian Federation.

6.2. CERTIFIER provides information across EAEU jurisdictions (Russia, Kazakhstan, Belarus, Kyrgyzstan, Armenia) without transferring personal data to those jurisdictions.
